# 更新日志

## [1.1.0] - 2026-02-12

### 新增功能

#### 天气组件增强
- 显示完整地区名（城市·省/市·国家）
- 添加未来三日天气预报
- 显示当前湿度和风速信息
- 优化组件尺寸，保持与搜索框高度一致

#### 导航栏增强
- 自动获取网站 favicon 图标
- 每行最多显示5个导航，自动换行
- 组件高度自适应，随内容动态扩展
- 支持从 Chrome 书签导入

#### GitHub Actions CI/CD
- 配置自动构建流水线
- 推送代码自动编译并创建 Release
- 自动生成可安装的扩展包

### 优化改进

#### UI/UX 优化
- 移除无用的语言切换按钮
- 搜索框和导航栏默认透明，悬停显示毛玻璃效果
- 热榜组件文字调大，更易阅读
- 右下角按钮垂直排列（壁纸设置、编辑布局）

#### 存储层优化
- 修复导航数据保存问题
- 兼容旧格式数据（带时间戳的对象格式）
- 使用 Chrome Storage Sync 持久化存储
- 添加详细的调试日志

### 修复问题
- 修复导航修改后无法保存的问题
- 修复空数组被误判为无数据的问题
- 修复首次加载时重复保存默认链接的问题

### 技术改进
- 增强类型检查和错误处理
- 优化数据结构兼容性
- 改进组件生命周期管理

## [1.0.0] - 2026-02-10

### 初始版本
- 动态壁纸（Bing 每日一图 + 自定义）
- 时钟和天气显示
- 多搜索引擎支持（百度、Google、Bing、DuckDuckGo）
- AI 模式（通义千问、Gemini、ChatGPT、Claude、Kimi）
- GitHub Trending、知乎热榜、V2EX 热议
- 自定义导航栏
- 可拖拽网格布局
- 透明度调节
- 毛玻璃设计风格
